# Advanced Java Concepts

## Files
- Generics.java - Generic classes, methods, bounded types
- ExceptionHandling.java - Try-catch, custom exceptions, throws
- Multithreading.java - Thread creation, synchronization

## Topics Covered
- **Generics**: Type safety, wildcards, bounded parameters
- **Exception Handling**: Built-in and custom exceptions
- **Multithreading**: Thread lifecycle, synchronization

## Run
```bash
javac Generics.java && java Generics
```
