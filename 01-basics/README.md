# Java Basics

## Files
- Hello.java - First program
- variables.java - Variable types  
- DataTypes.java - All primitive data types
- Operators.java - Arithmetic, logical, comparison operators
- Conditionals.java - If/else statements
- Loop.java - For/while loops
- Arrays.java - Array operations
- Strings.java - String manipulation
- Methods.java - Method creation and overloading
- InputOutput.java - Scanner for user input
- MultiVariable.java - Multiple variables

## Learning Order
1. Hello.java
