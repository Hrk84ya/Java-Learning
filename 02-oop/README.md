# Object-Oriented Programming

## Files
- OOPS.java - Classes and objects basics
- Constructors.java - Default, parameterized, copy constructors
- Encapsulation.java - Private fields, getters/setters
- Inheritance.java - Single inheritance
- multiInheritance.java - Multiple inheritance via interfaces
- Polymorphism.java - Method overriding, runtime polymorphism
- Abstraction.java - Abstract classes and methods
- Interfaces.java - Interface implementation, default methods
- StaticKeyword.java - Static variables, methods, blocks

## Learning Order
1. OOPS.java
2. Constructors.java
3. Encapsulation.java
4. Inheritance.java
5. Polymorphism.java
6. Abstraction.java
7. Interfaces.java
8. multiInheritance.java
9. StaticKeyword.java

## Run
```bash
javac OOPS.java && java OOPS
```
